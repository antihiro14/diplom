# Food Recognition System

## Опис
Цей проєкт є частиною бакалаврської роботи та спрямований на розпізнавання страв за зображеннями з використанням методів комп'ютерного зору.

## Функціональність
- Обробка датасету зображень страв
- Навчання моделі класифікації
- Визначення типу страви за фото

## Структура
- `src/` — логіка обробки даних та навчання
- `models/` — архітектура моделі
- `data/` — приклади даних (без повного датасету)
- `main.py` — точка входу

## Стан проєкту
Проєкт знаходиться на початковому етапі розробки.
